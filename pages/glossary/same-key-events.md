---
 title: Same key events
 key: same-key-events
 unambiguous: true
 objective: true
---

Two [keyboard events][] are _same key events_ if they have the same value for all the following attributes:
- `key`
- `code`
- `location`
- `repeat`
- `isComposing`

[keyboard events]: https://www.w3.org/TR/uievents/#events-keyboardevents
